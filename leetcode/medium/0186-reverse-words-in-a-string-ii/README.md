# 186. Reverse Words in a String II

**Difficulty:** ![Medium](https://img.shields.io/badge/Medium-orange)

**Link:** [LeetCode Problem 186](https://leetcode.com/problems/reverse-words-in-a-string-ii/)

## Problem Description

Given an input string as a character array, reverse the string word by word.

## Examples

### Example 1

**Input:** `["t","h","e"," ","s","k","y"," ","i","s"," ","b","l","u","e"]`

**Output:** `["b","l","u","e"," ","i","s"," ","s","k","y"," ","t","h","e"]`

## Constraints

- A word consists of non-space characters only
- The input has no leading or trailing spaces
- Words are separated by single spaces

## Follow-up

Could you do it in-place without allocating extra space?
