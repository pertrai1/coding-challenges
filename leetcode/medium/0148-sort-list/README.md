# [Sort List](https://leetcode.com/problems/sort-list) ![Shield](https://img.shields.io/badge/Medium-orange)

Given the `head` of a linked list, return \*the list after sorting it in **ascending order\***.

## Examples

**Example 1:**

![Image](https://assets.leetcode.com/uploads/2020/09/14/sort_list_1.jpg)

```bash
Input: head = [4,2,1,3]
Output: [1,2,3,4]
```

**Example 2:**

![Image](https://assets.leetcode.com/uploads/2020/09/14/sort_list_2.jpg)

```bash
Input: head = [-1,5,3,4,0]
Output: [-1,0,3,4,5]
```

**Example 3:**

```bash
Input: head = []
Output: []
```

## Constraints

- The number of nodes in the list is in the range `[0, 5 * 10⁴]`.
- `-10⁵ <= Node.val <= 10⁵`

## Follow up

Can you sort the linked list in `O(n logn)` time and `O(1)` memory (i.e. constant space)?
